# iOS-Style Calculator App

A playful, responsive calculator built as a hands-on learning project for exploring how to code with AI.

This project started as a simple browser-based calculator and grew into a polished, feature-rich app with keyboard support, animations, history, and responsive styling. It is a great example of how small, iterative steps can turn into something visually impressive.

## ✨ Features

- Responsive iOS-inspired calculator layout
- Clickable number pad and operator buttons
- Keyboard support for faster input
- Backspace and clear button behavior
- Press animations and sound effects
- Percentage handling that feels more like a real calculator
- Recent calculation history
- Display formatting that stays inside the screen
- Responsive layout for smaller screens and short browser windows

## 🧠 Why This Project Exists

This project is part of a learning journey for building web apps with AI-assisted coding.
It shows how ideas can be broken into small steps, tested, and improved over time.

## 🛠️ Technologies Used

- HTML
- CSS
- JavaScript

## ▶️ How to Run

Open the project in your browser by loading the file:

```bash
index.html
```

If you prefer, you can also serve the folder locally with a simple static server.
